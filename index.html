eINIT

eINIT used to be an alternate implementation of the /sbin/init programme for Linux and FreeBSD. Well, I suppose it still is, but the project has been on hold for years now. Some parts of the project seem to have been lost, but most of it has been moved right here to GitHub. Ping Magnus (@jyujin) if you'd like to see it revived.

To see all the projects, go to https://github.com/einit and have a browse.

The original content of the website, the blog posts and documentation items, have been archived at my blog: http://ef.gy/einit
